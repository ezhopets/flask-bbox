# flask-bbox

#### Start
```bash
$ pip3 install flask
```

```bash
$ pip3 install opencv-python
```

```bash
$ pip3 install pdfminer
```

```bash
$ pip3 install pdf2image
```

```bash
$ pip3 install Werkzeug
```

```bash
$ set FLASK_APP=app
```

```bash
$ flask run
```
or

```bash
$ flask run --host=0.0.0.0
```
